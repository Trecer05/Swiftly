# Request Handlers

Обработчики HTTP и WebSocket запросов.

## Подпакеты:
- `mobile/` - Специфичные эндпоинты для мобильного API
- `chat/` - Обработчики чат-функционала
- Список пакетов обновится в процессе разработки

Структура:
- Каждый файл соответствует группе эндпоинтов
- Использует сервисный слой для бизнес-логики
