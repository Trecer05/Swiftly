# Configuration

Хранение и загрузка конфигурации приложения (env-переменные, yaml-файлы).

## Основные функции:
- Загрузка настроек из файлов/окружения
- Валидация конфигурации
- Предоставление глобального доступа к настройкам

Пример использования:
```go
cfg := config.Load()
fmt.Println("HTTP port:", cfg.HTTP.Port)
```
