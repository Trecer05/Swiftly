# DOCS

Документации API микросервисов приложения Swiftly

## Подпакеты:
- `auth/` - Документация микросервиса аутентификации
- `chat/` - Документация микросервиса чата и сообщений
- `calls/` - Документация системы звонков (WebRTC)

## Обзор API

### Аутентификация
- **Базовый URL:** `/api/v1`
- **Методы:** POST /login, POST /register, POST /logout, POST /refresh
- **Безопасность:** JWT токены (access + refresh)
- **Документация:** [AUTH_API_DOC.md](auth/AUTH_API_DOC.md)

### Чат и сообщения
- **Базовый URL:** `/api/v1`
- **Функции:** Приватные чаты, групповые чаты, файлы, WebSocket
- **Real-time:** WebSocket соединения с Redis Pub/Sub
- **Документация:** [CHAT_API_DOC.md](chat/CHAT_API_DOC.md)

### Звонки (WebRTC)
- **Базовый URL:** `/api/v1`
- **Функции:** Аудио/видео звонки, групповые звонки
- **Технологии:** WebRTC, WebSocket сигналинг
- **Документация:** [CALLS_API_DOC.md](calls/CALLS_API_DOC.md)

## Технические детали

### Архитектура
- **Микросервисы:** Auth, Chat, Calls
- **База данных:** PostgreSQL с миграциями
- **Кэш:** Redis для real-time функций
- **Файлы:** Локальное хранение с организованной структурой

### Безопасность
- JWT аутентификация для всех эндпойнтов
- Валидация прав доступа к чатам и группам
- Изоляция WebRTC комнат по пользователям
- Защита от CSRF и других атак

### Производительность
- Redis Pub/Sub для real-time сообщений
- Оптимизированные SQL запросы с индексами
- Эффективная обработка файлов
- Масштабируемая архитектура WebRTC
